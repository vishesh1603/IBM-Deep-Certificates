**Sample Project**
- `Car Price Prediction/` — an ML pipeline that predicts used car prices from a dataset of 16,754 listings. Ridge Regression, Random Forest, Extra Trees, XGBoost, and LightGBM are trained in log-space and combined into an ensemble, reaching an R² of 0.96 and a mean absolute error of about $3,500. A single-page Power BI dashboard is included alongside the notebook.
